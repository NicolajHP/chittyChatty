How to use the program:
1: move into the server folder and type "go run server.go" to run the server
2: open another terminal and move into the client folder and type "go run client.go" to run the client
3: you can name a client with a command line flag by typing "go run client.go --name=insertNameHere"
4: connect multiple clients, current limit is unknown to us, but works as intended with at least 3 clients
5: type in the chat using the client running terminals!
6: to make a client LEAVE, simply type /leave in the chat
